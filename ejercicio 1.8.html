Ejercicios de: If-else y if-elseif-else JavaScript

Ejercicio # 08.

-Escribe un programa que calcule el índice de masa corporal (IMC) de una persona y le muestre un mensaje según el rango en el que se encuentre. Los requerimientos son: Solicitar al usuario su peso en kilogramos.usuario su altura en metros. Calcular el IMC utilizando la fórmula: IMC = peso / (altura * altura). Mostrar un mensaje según el IMC: IMC < 18.5: "Bajo peso". 18.5 <= IMC < 25: "Peso normal". 25 <= IMC < 30: "Sobrepeso". IMC >= 30: "Obesidad".
